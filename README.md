# Adopta - Comunidad de Ayuda Animal

**Adopta** es una aplicación móvil creada para apoyar la labor de una fundación de rescate animal. Permite reportar animales en situación de calle, coordinar voluntarios, gestionar hogares temporales y facilitar adopciones responsables.

## Características principales
- 📍 Reportar animales con foto, ubicación y nivel de urgencia.
- 🗺️ Mapa interactivo con casos activos.
- 👥 Registro de voluntarios y hogares temporales.
- 📋 Formulario de adopción integrado.
- 🔔 Comunicación en tiempo real (notificaciones push).

## Tecnologías utilizadas
- **Frontend:** Flutter (Dart) - una sola base de código para Android (APK) y posterior iOS.
- **Backend:** Firebase (Authentication, Firestore, Storage, Cloud Messaging).
- **Mapas:** Google Maps Flutter.
- **Control de versiones:** GitHub.
